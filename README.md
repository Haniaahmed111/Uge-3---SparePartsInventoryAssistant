SparePartsInventoryAssistant

Dette program er et lille tekst-program, hvor brugeren kan spørge efter reservedele.
Programmet viser, om delen findes på lager eller ej.

Hvad programmet gør:
- Det starter med at vise en velkomstbesked.
- Brugeren bliver spurgt: “Which part do you need?”
- Programmet sammenligner derefter det, brugeren skrev, med tre dele i lageret:
  - hydraulic pump
  - PLC module
  - servo motor
    
- Hvis brugeren skriver én af disse dele præcist, svarer programmet, at delen findes og afslutter.
- Hvis brugeren skriver noget andet, svarer programmet, at delen ikke findes.
- Hvis brugeren spørger:
  - “Do you actually have any parts?”
  - “Is there anything in stock at all?”
    
så viser programmet alle tre dele.
